# react-redux-basic-starter
Basic app in React and Redux & Redux logger.

It uses `create-react-app`

## Using NPM

#### Run the app

```
npm install

npm start
```
#### Build for deployment
```
npm run build
```
__

## Using Yarn

```
yarn install

yarn start
```
#### Build for deployment
```
yarn run build
```

open http://localhost:3000/
